Problem Link: https://leetcode.com/problems/remove-duplicates-from-sorted-array-ii/



## Solution I

```python=
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        i = 0
        k = 2
        for num in nums:
            if i < k or num != nums[i-k]:
                nums[i] = num
                i += 1
        return i
```

#### Complexity Analysis:
- Time: O(n)
- Space: O(1)
