## Solution I
binary search

```python=
class Solution:
    def minEatingSpeed(self, piles: List[int], h: int) -> int:
        left, right = 1, max(piles)
        while left < right:
            middle = (left + right) // 2
            hour_spent = 0
            for pile in piles:
                hour_spent += (pile + middle - 1) // middle
            if hour_spent <= h:
                right = middle
            else:
                left = middle + 1
        return right
```
