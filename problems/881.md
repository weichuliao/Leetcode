## Solution I
greedy, two pointer

```python=
class Solution:
    def numRescueBoats(self, people: List[int], limit: int) -> int:
        people.sort()
        i, j, ans = 0, len(people) - 1, 0
        while i <= j:
            ans += 1
            if people[i] + people[j] <= limit:
                i += 1
            j -= 1
        return ans
```

#### Complexity Analysis:
- Time: O(nlogn)
- Space: O(n)
